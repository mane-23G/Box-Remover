Extension that essentially removes the boxes on the physics answer textbook automatically.

How to install on Firefox

1. download the directory
2. Got to firefox about:debugging#/runtime/this-firefox
3. Click "Load Temporary Add-On"
4. Go to the directory downloaded and click on any file in the directory
5. Go to giancolianswers and see - or should I say no longer see - the box

How to install on Chrome 
1. Download the directory
2. Go to chrome://extensions/
3. Click "Load unpacked"
4. Select the downloaded directory and click on the directory
5. Go to giancolianswers and see - or should I say no longer see - the box
